# SQL Syntax Checker
A real-time web-based SQL query validator built with HTML, CSS, JavaScript, Python, and Bootstrap.

## Features
- Validates SQL queries as you type
- Instant feedback for 20+ SQL syntax patterns
- Intelligent error suggestions

## Tech Stack
HTML · CSS · JavaScript · Python · Bootstrap

## How to run
Open index.html in a browser.
